## 快速使用

```bash
# 安装脚手架
$ pnpm add @fai-hd/hd-sfc-cli -D

# 添加scripts
$ code package.json

"scripts": {
  "dev": "hd-sfc-cli dev",
  "build": "hd-sfc-cli build"
}

# 开发模式
$ pnpm build

# 开始打包
$ pnpm build
```
